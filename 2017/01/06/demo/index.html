Hexo+Github写博客

2017.01.06 Xie Yiming  热度 ℃

一、首先建立一个工程文件夹，在该文件夹下打开命令窗口

1. 使用npm安装hexo

npm install -g hexo

2. 初始化hexo

hexo init blog
npm install

3. 生成静态页面

hexo generate

4. 启动本地服务，进行文章预览调试

hexo serve
ctrl+c退出服务

5. github上新建一个仓库

6. 修改_config.yml文件（root:/blog/）

7. 下载安装hexo-deployer-git

npm install hexo-deployer-git --save

8. 发布到github

hexo deploy --generate

注意：重新发布
hexo clean
hexo deployer --generate或者hexo d g
