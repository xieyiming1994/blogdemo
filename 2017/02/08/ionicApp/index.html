一、JAVA环境

1.想要成功配置java的环境变量，那肯定就要安装JDK，才能开始配置的。

2.成功安装之后，进行测试是否真的成功安装，点击【开始】—-【运行】—-输入 CMD，在命令提示符里面输入“Java -version”并按回车键，出现下图，即为安装成功。

3.下面开始配置环境变量，右击【我的电脑】—【属性】—–【高级】—【环境变量】。

选择【新建系统变量】–弹出“编辑用户变量”对话框，在“变量名”文本框输入“JAVA_HOME”,在“变量值”文本框输入JDK的安装路径：

单击“确定”按钮，如图：

4.在“系统变量”选项区域中查看PATH变量，如果不存在，则新建变量 PATH，否则选中该变量，单击“编辑”按钮，在“变量值”文本框的起始位置添加“%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;”或者是直接“%JAVA_HOME%\bin;”，单击确定按钮，如图：

5.在“系统变量”选项区域中查看CLASSPATH 变量，如果不存在，则新建变量CLASSPATH，否则选中该变量，单击“编辑”按钮，在“变量值”文本框的起始位置添加“.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;”。如图：

6.现在测试环境变量的配置成功与否。在DOS命令行窗口输入“JAVA”，输出帮助信息即为配置正确。如图：

7.现在测试环境变量的配置成功与否。在DOS命令行窗口输入“JAVAC”，输出帮助信息即为配置正确。如图：

记得不要漏掉最后的“;”符号

###三、App工程的创建、启动和应用
1.新建一个文件夹，再在文件夹下创建工程myprj，如下图：

2.在此目录下打开命令提示框，输入命令：
ionic start myapp tabs //使用这个命令创建一个App工程

3.进入myprj,开始如下操作：

cd myprj     //进入myprj文件下
ionic serve   //使用这个命令启动应用，会自动在浏览器中打开应用
ionic platfrom add android     //为工程添加平台编译选项
ionic platfrom add android     //为工程添加平台编译选项
ionic build android     //编译
